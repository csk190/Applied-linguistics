## 📅 Week 06 Exercise Guide (4/15 ~ 4/21)
#### **Topic: Coding4ET Exercises**

이번 주차 연습문제를 풀고 본인의 GitHub 레포지토리에 과제를 제출하는 방법입니다. 아래 단계를 순서대로 따라하세요.

---

### **Step 1. 본인의 과제 제출용 레포지토리 생성**
* GitHub 계정에 로그인한 후, **S2026**이라는 이름의 새로운 레포지토리(Repository)를 생성합니다. 
* 이 공간은 이번 학기 여러분의 모든 코딩 결과물이 저장될 포트폴리오가 됩니다.

### **Step 2. 연습문제 열기 (Colab 연동)**
* [Coding4ET GitHub 링크](https://github.com/MK316/Coding4ET/blob/main/README.md)로 접속합니다.
* 연습문제 중 EX3.5 파일을 연 후, 페이지에 있는 **`Open in Colab`** 버튼을 클릭하여 코랩 환경에서 문제를 엽니다.

### **Step 3. 코드 완성 및 실행 (DIY Sections)**
* 연습문제 내의 설명과 예시 코드를 충분히 학습합니다.
* 지시에 따라 **DIY code sections**의 빈칸을 채워 코드를 완성합니다.
* **중요:** 모든 코드 셀을 실행하여 결과(그래프, 텍스트 등)가 화면에 올바르게 출력되는지 확인합니다.

### **Step 4. 저장 설정 변경 (결과값 보존)**
저장된 파일에 여러분의 실행 결과가 포함되도록 설정을 반드시 변경해야 합니다.
1. 상단 메뉴에서 **수정(Edit) > 노트북 설정**(Notebook settings)을 클릭합니다.
2. **이 노트북을 저장할 때 코드 셀 출력 제외(Omit code cell output...)** 항목의 **체크를 해제**합니다.
3. **[저장(Save)]** 버튼을 누릅니다.

### **Step 5. 본인의 레포지토리에 저장**
1. 메뉴에서 **`파일(File) > GitHub에 복사본 저장(Save a copy in GitHub`**)을 선택합니다. 
2. 대상 레포지토리(Repository)를 여러분이 만든 **S2026**으로 선택합니다.
3. **파일명(File name)은 원래 연습문제 파일명에 본인 이름 이니셜 3글자를 추가하여** 저장합니다. (예: EX3.5_KMR.ipynb)

---
⚠️ **주의사항:** 설정을 변경하지 않고 저장하면 실행 결과가 없는 빈 코드만 제출되어 감점 요인이 될 수 있습니다. 반드시 **Step 4**를 확인하세요!
